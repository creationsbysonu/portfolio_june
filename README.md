# Senior Portfolio Website

Minimal React + TypeScript + Vite portfolio site built around a refined design system, accessible motion, and high-trust product storytelling.

## Stack

- React 19
- TypeScript
- Vite
- CSS variables and responsive layout primitives

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run lint` runs Oxlint.

## Notes

- The site uses local scroll-triggered reveals and counter animations.
- Motion respects `prefers-reduced-motion`.
- Copy and project content are placeholder-driven and should be replaced with real portfolio details.
